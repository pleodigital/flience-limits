# Users Limit plugin for Craft CMS 3.x

Limit user to download 20 wallpapers per month

![Screenshot](resources/img/plugin-logo.png)

## Requirements

This plugin requires Craft CMS 3.0.0-beta.23 or later.

## Installation

To install the plugin, follow these instructions.

1. Open your terminal and go to your Craft project:

        cd /path/to/project

2. Then tell Composer to load the plugin:

        composer require https://github.com/TrebuhS/flience-limits/users-limit

3. In the Control Panel, go to Settings → Plugins and click the “Install” button for Users Limit.
